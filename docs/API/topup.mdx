Gunakan endpoint ini untuk melakukan **topup saldo pengguna**.

---

## 🧭 Endpoint
| Atribut | Nilai |
|----------|-------|
| **URL** | `https://api.hesdapay.com/v1/topup` |
| **Method** | `POST` |
| **Auth** | Token Bearer (`Authorization: Bearer <API_KEY>`) |
| **Content-Type** | `application/json` |

---

## 📥 Parameter

| Nama | Tipe | Wajib | Keterangan |
|------|------|--------|-------------|
| `user_id` | `string` | ✅ | ID unik pengguna |
| `nominal` | `number` | ✅ | Nominal topup dalam rupiah |
| `metode` | `string` | ✅ | Jenis metode pembayaran (`bank`, `qris`, `ewallet`) |
| `reff_id` | `string` | ❌ | Kode unik referensi transaksi (opsional, auto generate jika kosong) |

---

## 📘 Contoh Request

```bash
curl -X POST https://api.hesdapay.com/v1/topup \
-H "Content-Type: application/json" \
-H "Authorization: Bearer 1234567890abcdef" \
-d '{
  "user_id": "USR001",
  "nominal": 50000,
  "metode": "qris"
},
```
